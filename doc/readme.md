## Installation

To install zpy, you need Python 3.4 or later.

### Debian 8, Ubuntu 14.04 or later

    # apt-get install python3-pip
    # pip3 install zpy

### Fedora 22

    # dnf install python3-pip
    # pip3 install zpy

### OS X (Homebrew)

    $ brew install python3
    $ pip3 install zpy
